Anchor Tag - <a>


It is used to create:

  • Links to webpages.
  • Links to files.
  • Links to sections within the pages.
  • Emails,phone call,download

href - Hypertext Reference

Specifies Destination of the link.


target Attribute

target="_blank" - Opens link in new Tab

target="_self" - Opens link in same Tab


download

downloads file

<a> href="file.pdf" download <a>


Email

opens email app

<a> href="mailto:test@gmail.com" Send mail <a>


phone

used to call

<a> href="tel:+91999999" Call us <a>


rel Attribute in HTML

The rel attribute describes the relationship between the current page and the linked resource.


rel helps with security,privacy,SEO,performance.



Problem Without rel


when a tag opens a new tab , the new page can access your original page using JS. Which typically happens through window.opener, This is called Reverse Tabnabbing


Solution - noopener


rel="noopener" - Prevents new page from accessing window.opener


With noopener - window.opener = null


Without noopener - window.opener = original tab


noreffer - Does two things


It hides referrer info. When user click a link , the Destination web receives info like which page user came from and URL of page.

This is called referrer info.

Norefferer solves this so destination does not know where user came from.


It also behaves as noopener

Norefferer also blocks window.opener
