Box Model

CSS BOX MODEL

Every HTML element is treated like a rectangular box.


A box contains:

  1. content
  2. padding
  3. border
  4. margin


Parts of box model


Content

Actual image/text inside element.

examples:
width:200px;
height:100px;



Padding

Space INSIDE the element around content.

examples:
padding:20px;



Border

Boundary around padding/content.

examples:
border:2px;



Margin

space outside element

examples:
margin:20px;




Important


By default css width only controls the content area.


Default behavior is
box-sizing: content-box;
which means: Final width = content + padding + border.


VISUAL BREAKDOWN

| border | padding | CONTENT(200px) | padding | border |

problem is :

suppose you have width:50% and padding 20px;
Now element bigger than 50%.

ENTER box-sizing:border-box

.box{
width:200px;
padding:20px;
border:10px solid black;
box-sizing:border-box;
}
